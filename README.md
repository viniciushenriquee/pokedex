# 🔵 Pokédex - Projeto API Pokémon

Uma Pokédex moderna e responsiva desenvolvida com HTML, CSS e JavaScript puro (Vanilla JS), consumindo dados da [PokéAPI](https://pokeapi.co/).

![Pokédex Preview](https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/25.png)

## 🚀 Funcionalidades

- **Tela de Início Dinâmica:** Uma interface de entrada estilizada com animações e botão para iniciar a jornada.
- **Busca em Tempo Real:** Pesquise seu Pokémon favorito pelo **nome** ou pelo **ID**.
- **Navegação Intuitiva:** Botões "Anterior" e "Próximo" para explorar a lista de Pokémon sequencialmente.
- **Design Responsivo:** Interface adaptável para diferentes tamanhos de tela (Desktop e Mobile).
- **Estética Temática:** Visual inspirado no universo Pokémon com fundo estilo "Comic/Quadrinhos" e paleta de cores azulada.
- **Dados Detalhados:** Exibe imagem oficial, nome, ID, altura, peso e tipos de cada Pokémon.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica do projeto.
- **CSS3:** Estilização avançada, animações de `bounce` e `fade-in`, e layout flexível.
- **JavaScript (ES6+):** Manipulação de DOM, eventos e consumo de API assíncrona (`Fetch API` / `Async-Await`).
- **PokéAPI:** Fonte de dados oficial para as informações dos Pokémon.

## 📦 Como executar o projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/pokedex.git
   ```
2. **Navegue até a pasta do projeto:**
   ```bash
   cd pokedex
   ```
3. **Abra o arquivo principal:**
   Basta abrir o arquivo `src/index.html` em qualquer navegador moderno.

O projeto conta com:
- **Primary Color:** `#3b4cca` (Azul Pokémon)
- **Background:** Imagem abstrata de quadrinhos.
- **Fontes:** Roboto e Comic Neue (via Google Fonts).
